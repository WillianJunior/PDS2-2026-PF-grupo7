# PDS2-2026-PF-grupo7

Compilação

$ make

Atenção: Criar antes um diretório ./build/
Execução

$ make run

Compilação Testes

$ make tests

Execução Testes

$ make run_tests

Relatório de Cobertura

Para imprimir o relatório simplificado no próprio Terminal, utilizar:

$ make coverage

Para gerar o relatório em HTML, utilizar:

$ make html_coverage
